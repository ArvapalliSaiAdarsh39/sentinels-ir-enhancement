# Infrared Image Colourization and Enhancement Engine

A deterministic, metadata-safe satellite data processing suite built for the Bharatiya Antariksh Hackathon 2026. This application applies explicit pixel-based mathematical operations—such as Contrast Limited Adaptive Histogram Equalization (CLAHE) and Laplacian edge filters—to convert single-channel monochrome IR telemetry into vivid, high-contrast pseudo-RGB visualizations. 

Designed to ensure 100% spatial accuracy, zero coordinate drift for reliable object interpretation in adverse low-light scenarios.
